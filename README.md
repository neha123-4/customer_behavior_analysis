# customer_behavior_analysis
data analytics project showcasing customer behavior analysis using python , sql, Power BI.

1. Overview

This project demonstrates an end-to-end data analysis workflow — from raw dataset to business insights and executive presentation.

The project covers:

Loading and analyzing data using Python

Performing Exploratory Data Analysis (EDA)

Cleaning and preparing data for analysis

Running analytical queries in MySQL

Building an interactive Power BI dashboard

Creating a detailed analytical report

Designing a presentation using Gamma

The goal of this project is to showcase practical data analysis skills, SQL proficiency, dashboard design, and the ability to communicate insights clearly.
